## Estructura de Carpetas

- 📁 **assets/**: Contiene todas las imágenes, capturas de pantalla y archivos de mapeo multimedia.
- 📁 **css/**: Contiene la hoja de estilos principal (`main.css`).
- 📁 **js/**: Contiene la lógica interactiva en JavaScript (`main.js`).
- 📁 **wp-screenshots/**: Capturas de pantalla del estado actual en WordPress.
- 📄 **index.html**: El punto de entrada principal del sitio web.

## Cómo Ejecutar
Simplemente abre el archivo `index.html` en cualquier navegador moderno. Todos los recursos se cargarán automáticamente desde las carpetas correspondientes.
